PROJECT PRIVIEW

기존에 있던 웹사이트의 장점을 살리고 본질을 해치지 않는 선에서 소비자의 입장에서 생각하여 더욱 편리한 방향으로 리디자인 하려고 합니다.

PROJECT GOAL

PROBLEM

룩넌페이지는 이미지 회전이 빠른편입니다. 설명부분을 이미지에 마우스를 올려야 보이게끔 해두어 사용자의 불필요한 동작이 한번 더 요구됩니다.

이미지 표현이 너무 많아 사이트가 전반적으로 복잡하고 어지러움.

SOLUTION

이미지 하단에 설명을 바로 정렬해 사용자의 불필요한 동작을 방지합니다.

대표적인 이미지들을 취합하여 슬라이드로 정렬합니다.

KEYWORD

룩넌의 특성과 브랜드 컬러, 방문자들의 의견을 리서치하여 키워드를 도출했습니다.
